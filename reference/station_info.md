# AZMet station names and locations

Station names, station IDs, and location (lat, lon, elevation) of active
AZMet weather stations

## Usage

``` r
station_info
```

## Format

A tibble with 30 rows and 5 columns:

- `meta_station_name`:

  Station name

- `meta_station_id`:

  Station ID

- `latitude`, `logitude`:

  Station location

- `elev_m`:

  Station elevation in meters
